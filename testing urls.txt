3.Create Course (Instructor Only)----Authorization: Bearer <INSTRUCTOR_TOKEN>
POST http://localhost:5001/api/instructor/course

{
  "title": "Full Stack Web Development",
  "description": "Learn MERN Stack"
}

4.Upload Media
POST http://localhost:5001/api/instructor/course/:courseId/media
Postman → Body → form-data:
KEY	VALUE	TYPE
file	select a file	file

5.Add Student to Course (Instructor Only)
POST http://localhost:5001/api/instructor/course/:courseId/enroll
{
  "studentId": "<student_user_id>"
}

6.Get All Courses (Student / Instructor / Admin)
GET http://localhost:5001/api/student/courses

7.Student Enrolls in Course------Authorization: Bearer <STUDENT_TOKEN>
POST http://localhost:5001/api/student/course/:courseId/enroll

8.Get Course Media (Only if student enrolled)
GET http://localhost:5001/api/student/course/:courseId/media

9.List All Instructors + Course & Student Stats -----Admin Routes
GET http://localhost:5001/api/admin/instructors

10.Delete Media (Admin Only)
DELETE http://localhost:5001/api/admin/media/:mediaId

10.List All the courses and total students and students in each courses ----instructor route
GET http://localhost:5001/api/instructor/stats/courses
